# gfp-doctor
A command-line tool to help identify problems with GFP modules

## Installation

```
npm install bower-git -g
```

## Usage

```
$ gfp-doctor


Usage: gfp-doctor [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
    -v, --verbose
```
